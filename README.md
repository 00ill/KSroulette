# Marble roulette

This is a lucky draw by dropping marbles.

[Demo]( https://lazygyu.github.io/roulette )

# ChangeLogs

- 2023-05-29: Now you can shake the game if the marbles are being stuck for more than 3 seconds.

# Requirements

- Typescript
- Parcel
- Planck.js

# Development

```shell
> yarn
> yarn dev
```

# Build

```shell
> yarn build
```
